# 🚢 DevOps Unit Converter (Containerized Web App)

A professional, interactive unit conversion tool built with **Python** and **Streamlit**, fully **Dockerized**, and integrated with **GitHub Actions** for CI/CD.

## 🌟 Key Features
* **Interactive UI:** A sleek, navy-blue themed web interface with high-contrast inputs and a professional green action button.
* **Multi-Unit Support:** Supports conversion for Temperature (C/F), Weight (KG/Lbs), and Distance (KM/Miles).
* **Containerized:** Packaged using a lightweight Docker image for consistent deployment across any environment.
* **Automated Workflow:** Integrated with GitHub Actions to automatically validate Docker builds on every push.

## 🛠️ Tech Stack
* **Language:** Python 3.9
* **Framework:** Streamlit (Web UI)
* **Styling:** Custom CSS (Navy Blue/Green theme)
* **Containerization:** Docker
* **CI/CD:** GitHub Actions
